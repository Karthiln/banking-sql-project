# 💰 Banking Transaction System (SQL Project)

## 📌 Overview
This project simulates a banking system using SQL with transaction handling and fraud detection.

## 🧰 Tools Used
- MySQL Workbench
- SQL

## 🔧 Features
- Money transfer using transactions
- Rollback for failure handling
- Fraud detection for high-value transactions

## 💡 Concepts Covered
- Transactions (COMMIT, ROLLBACK)
- ACID properties
- SQL queries and joins
